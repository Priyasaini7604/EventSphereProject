# 🎓 EventSphere – University-Wide Event Management Platform

EventSphere is a centralized, mobile-responsive full-stack web application designed to manage intra-college and inter-college events across universities. It streamlines the entire event lifecycle into a single unified system.

## 🔗 Repository

[EventSphere GitHub Repository](https://github.com/Priyasaini7604/EventSphereProject)

---

## 🚩 Problem

University event communication is scattered across WhatsApp, emails, and social media, leading to missed updates, fake registrations, and poor attendance tracking.

---

## 💡 Solution

EventSphere provides a role-based, secure platform that manages event proposals, approvals, registrations, live attendance, analytics, and certificate generation in one place.

---

## ✨ Features

- Role-based dashboards for Students, Clubs, and Admins
- Event creation and approval workflow
- Student registration and notifications
- 📍 Live attendance system to prevent fake registrations
- 🎓 Certificates for verified attendees only
- 📊 Participation and engagement analytics
- 🤖 AI-based personalized event recommendations
- Fully mobile-responsive UI
